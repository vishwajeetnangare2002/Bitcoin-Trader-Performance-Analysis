# 📈 Bitcoin Trader Performance Analysis

## Project Overview

This project analyzes Hyperliquid trader performance by combining historical trading data with the Bitcoin Fear & Greed Index.

The objective is to understand how market sentiment influences trading behavior and profitability.

---

## Problem Statement

Can Bitcoin market sentiment (Fear/Greed) explain differences in trader performance?

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## Dataset

### Dataset 1
Bitcoin Fear & Greed Index

Contains:
- Date
- Sentiment Value
- Classification

### Dataset 2
Hyperliquid Historical Trader Data

Contains:
- Account
- Coin
- Buy/Sell
- Closed PnL
- Fees
- Timestamp

5. Data Visualization

7. Recommendations

## Key Visualizations

- Market Sentiment Distribution
- Profit Distribution
- Profit by Market Sentiment
- Buy vs Sell Analysis
- Coin-wise Profit Analysis
- Top Traders
- Daily Trading Activity
- Correlation Heatmap

---

## Business Insights

- Market sentiment impacts trader performance.
- Extreme Greed periods show different trading characteristics than Fear periods.
- Coin-wise profitability varies significantly.
- Win/Loss analysis highlights trading effectiveness.
